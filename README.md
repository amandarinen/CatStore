# CatStore
 
A web-based cat store built as part of my course in interface development. Browse cats fetched from a live API, add them to your cart, and checkout.
 
## Features
 
- Browse cats with breed name and country of origin
- Search by breed or country
- Pagination (10 cats per page)
- Add cats to cart (saved in localStorage)
- Checkout form with name, email and shipping address

## Tech Stack
 
- HTML, CSS, JavaScript
- [The Cat API](https://thecatapi.com/)
- Font Awesome icons
- VS Code + Live Preview

## Pages
 
| Page | Description |
|------|-------------|
| `HomePage.html` | Landing page |
| `CatInfo.html` | Browse and search cats |
| `OwnerInfo.html` | About us |
| `Cart.html` | View cart and checkout |
 
## Getting Started
 
1. Clone the repo
2. Open the project folder in VS Code
3. Start Live Preview or Live Server
4. Open `HomePage.html`
